codeREM

memory consolidation while you sleep

Your AI agent's memory decays every day. Dead links, stale facts, broken references. codeREM runs three phases of analysis on your Claude Code memory files and fixes what it finds. Every night. Automatically.

// the problem

Claude Code stores persistent memory in markdown files. These files accumulate silently. After a few weeks, your agent is making decisions based on information that's no longer true.

Dead file paths. Dates that say "last Thursday" from six weeks ago. Two files that contradict each other. Your agent doesn't know -- it trusts everything equally.

// the three phases

During REM sleep, your brain replays the day's experiences, strengthens useful connections, and prunes what doesn't matter. codeREM does the same thing.

PHASE 1

Surface Cleanup

  • Dead path detection
  • Relative date conversion
  • Frontmatter normalization
  • Duplicate detection
  • Trailing whitespace
PHASE 2

Structural Integrity

  • Cross-reference validation
  • Pointer file verification
  • Index consistency
  • Canonical source alignment
  • Orphan file detection
PHASE 3

Semantic Verification

  • Fact vs. current state
  • Contradiction detection
  • Staleness scoring
  • Temporal coherence
  • Recommended deletions

Each phase gets progressively deeper. Phase 1 fixes what it can automatically. Phase 2 checks structure. Phase 3 only flags -- it never auto-edits at semantic depth. Every change is git-committed so you can diff or rollback.

// what it produces

Dream Journal

Timestamped log of every fix, flag, and skip across all three phases. Your audit trail. See exactly what changed and why.

Git History

One commit per phase. Diff any run, rollback any change. Your memory directory becomes a versioned system, not a pile of files.

Chronicle Episodes

Autobiographical short stories generated from the day's findings. Llama writes outlines, Claude writes prose. A book, one episode per night.

Illustrations

SDXL Turbo on a local GPU. 0.3 seconds per image. Five art styles from Silverstein to stencil. Zero cost after hardware.
